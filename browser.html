Lesson 1

Introduction to CSS!

In the first lesson, we were introduced to CSS for the first time. We were enlightened on its meaning and why we should use CSS. Also, Ms. Grace taught us about the layers of a webpage. She told us the 3 different layers: the content, presentation, and behavior layer and how they work and function together to make up a webpage. After learning about the layers one-by-one, we were finally enlightened on the benefits, capabilities, and advantages of using CSS in our work.

Lesson 2

CSS Stylesheets!

In the next lesson, Ms. Grace further explained CSS. She told us about its structure and how the selector, property, and value work together. And of course, the types of stylesheets werr introduced to us. After learning about the 3 types: Embedded, Internal, and External Stylesheet, we were then taught how to code them and apply each. We were taught each step carefully, from using the style tag to creating a CSS file. Everything was clear.

Lesson 4

Classes!

Lesson 4 was short. First, we were introduced to the CSS Rule. It has 2 main parts: a selector and one or more style declarations. Afterwards, it was explained to us how classes work and how they are denoted by a period (.). Then to further expand our knowledge, Ms. Grace taught us about Class Selectors work. It is when an element and a selector was put together. This was to specify which element you want to style. Lastly, we were introduced to the principle of grouping selectors. Grouping selectors with the same style declaration makes things easier by avoiding repeating declarations, and making things more organized.

Lesson 5

Selectors!

This lesson was pretty simple. CSS Selectors are used to find or select the HTML Elements you want to style. Based on the lesson, there are 6 types: Class Selector, Element Selector, ID Selector, Universal Selector, Attribute Selector, and Group Selector. Each selector is different from the others. And there are certain times and circumstances on when certain selectors are to be used. Also, Ms. Grace showed us how the different selectors worked by coding them and seeing the output.

Lesson 6

Setting Dimensions!

The last lesson, Lesson 6, was all about setting dimensions and pseudo-classes, and links. CSS can be used to manage dimensions like the visibility and the width, and height of an HTML element on a webpage. As for the pseudo-classes, they have 4 types: link, visited, hover, and active. The link pseudo-class is for when the link is not visited yet. The visited pseudo-class is for when the link has been visited. The hover pseudo-class is for when the user is hovering the link. And the active pseudo-class is for when the user currently has the link opened. And that basically compromises the last lesson.
